
# 📅 Orbit | Smart Schedule Planner

Orbit is a modern, browser-based habit tracker and schedule planner. It helps users build consistency by organizing daily tasks into visual "streaks" and challenges.

## ✨ Features
- **Modern UI:** Built with a "Glassmorphism" design using Tailwind CSS.
- **Dynamic Challenges:** Create schedules for any number of days and tasks.
- **Progress Tracking:** Visual progress bars for every goal you set.
- **Auto-Save:** Uses browser `LocalStorage` so your data stays even if you refresh the page.
- **Fully Responsive:** Works perfectly on mobile, tablet, and desktop.

## 🛠️ Built With
- **HTML5** & **CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS** (for styling)
- **FontAwesome** (for iconography)

## 📖 How to Use
1. **Login:** Enter your name to create your local profile.
2. **Create:** Define your goal, set the number of days, and list your daily tasks.
3. **Track:** Open your schedule daily and mark tasks as "Done."
4. **Complete:** Watch your progress bar grow as you build your streak!
